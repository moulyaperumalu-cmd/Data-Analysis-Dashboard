# Blinkit Grocery Sales Analysis Dashboard

## Project Name :
Blinkit Grocery Sales Analysis Dashboard using Power BI.

## Project Objective :
The main objective of this project is to analyze Blinkit's grocery sales data and build an interactive Power BI dashboard that provides meaningful business insights. The dashboard helps stakeholders understand sales performance, customer preferences, outlet performance, and product distribution to support data-driven business decisions.

## Dataset Used :
Dataset Name: Blinkit Grocery Dataset
This dataset contains detailed information related to Blinkit's grocery sales and outlet performance. It includes the following attributes:
•	Item Identifier 
•	Item Type 
•	Item Fat Content 
•	Item Weight 
•	Item Visibility 
•	Sales 
•	Customer Rating 
•	Outlet Identifier 
•	Outlet Establishment Year 
•	Outlet Size 
•	Outlet Location Type 
•	Outlet Type 
The dataset is ideal for performing retail sales analysis, identifying business trends, evaluating outlet performance, and creating interactive dashboards using Power BI.
